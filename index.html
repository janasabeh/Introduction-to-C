Variables

Variables are names given to a storage area. This allows the program to access and manipulate the data inside this storage area.

They are used to store values or data that will be handled or used later on. They can store integers, decimals, and even characters depending on their data type.

Data Types

Data types refer to the type of data that can be stored in this variable. There are 4 main data types in C:

  • int: integers
  • float: decimal numbers
  • double: decimal numbers with double precision
  • char: one character represented inside single quotations ( ' ' )

Another type are strings: a sentence represented inside double quotations ( " " ).
More advanced data types may be arrays, and structs.

Variable declaration is informing the compiler of the variable's name & data type.
Variable initialization is assigning a value to this variable

Declaration & Initialization:

int x = 1;
float y = 1.5;
char z = 'A';
float t = x + y; Now t will contain 1+1.5 = 2.5
char str[] = "How are you." This is a declaration of a string

Characters are stored according to ASCII values.

Now you know how to declare and initialize variables. But what if i want to get the value from the user or print it on the screen?

Input/Output

The input/output functions come from the stdio (standard input ouput) library. Include it in your code by adding:
#include <stdio.h>
at the top of your file. This will allow you to use all the pre-defined functions in this library, such as printf and scanf.

Printf

printf is a function that takes variable or a string and prints it on the screen. The "f" stands for formatted: as you can format the string inside the printf. Example:

printf("Hello World!"); Output is: Hello World!
printf("Hello %i!",x); Output is: Hello 1!

The %i are format specifiers used to identify the type of variable being printed.Some of the format specifiers in C are:

  • %c: a single character
  • %s: a string
  • %i: an integer
  • %f: a float
  • %lf: a double

Scanf

scanf is a function that prompts the user to enter a value on the keyboard, and then stores it in a variable. Depending on the format specifier used, the scanf can read integers, floats, strings, doubles, or characters.Example:

scanf("%i",&x); Storing the integer in x.
scanf("%c",&z); Storing the character in z.
scanf("%s",&str); Storing the string in str
Main Function

The main function in C, is what the compiler executes when run. Without it, execution will fail.
The main function takes no arguments and returns an integer, usually 0 to indicate that the program finished successfully. It is written as:

int main()
{
 Coding statements
 return 0;
}
